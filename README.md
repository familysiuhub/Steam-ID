# Steam-ID Skills
This repository contains high-fidelity technical bridges for the Steam Web API, optimized for the Sovereign Stack.

## Contents
- skills/steam_web_api_v1.json: The core system_skill artifact.
- specs/steam_web_api_bridge_spec_v1.json: Technical specification for the bridge.

## Usage
Minds can interpret the steam_web_api_v1.json artifact to load the Steam_Web_API_v1 skill playbook. This skill supports Player Intelligence, Achievement Forensics, and Game News Pulses.

## Auth
Requires a valid steam.api.key stored in the technical vault.
